帮助与功能说明

1. 基础对话

  • 发送消息：在底部输入框中输入内容，点击右侧发送按钮或按 Enter 发送。
  • 欢迎卡片：首页建议卡片可以直接点击，快速开始一段新对话。
  • 提示词快捷栏：输入框上方支持快速切换全局提示词和当前提示词。

2. 侧边栏与历史

  • 新聊天：点击左侧“新聊天”会清空当前会话，开始新的对话。
  • 搜索历史：可以在侧边栏搜索框中实时筛选历史记录。
  • 折叠侧边栏：点击左上角图标可以收起或展开侧边栏。

3. 模式与附件

  • Ask / Agent：左侧胶囊按钮可以切换普通问答和更主动推进任务的模式。
  • 上传附件：点击输入框左侧 + 可上传文件、图片，支持多选。
  • GitHub 导入：也可以从同一个菜单中导入 GitHub 仓库链接。

4. 高级设置

点击左下角头像，选择“设置”即可打开高级配置。

  • 提示词：支持维护全局提示词和仅当前对话生效的提示词。
  • API 设置：支持手动添加模型，也支持解析 curl / Python 配置。

5. 数据存储

  • 目录选择：可以在“设置 - 存储设置”里切换数据保存目录。
  • 持久化：聊天记录、上传附件和索引数据都会保存在你选择的位置。
  • 自动同步：开启后会持续把对话内容写入存储目录。
