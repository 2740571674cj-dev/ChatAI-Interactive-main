帮助与功能说明

1. 基础对话交互

  • 发送消息：在底部输入框输入问题，点击右侧箭头或按 Enter 键即可发送消息。
  • 流式输出与动画：发送消息后，AI 头像会呈现“呼吸收缩”的动画，模拟思考或加载状态；在文字完全生成结束后，动画会自动停止。
  • 消息删除：将鼠标悬停在对话气泡上，旁边会显示垃圾桶图标。点击后该条消息将平滑删除。如果删除了当前屏幕上的所有消息，页面将自动重置回欢迎界面。
  • 快捷指令：在欢迎界面，点击提供的建议问题卡片，即可快速发起提问。
  • 提示词快捷控制：输入框上方设有全局和特定提示词快捷栏。点击可展开悬浮选择面板，直接勾选需生效的提示词，无需进入繁琐的设置页面。

2. 侧边栏与历史记录

  • 新聊天：点击侧边栏顶部的“新聊天”按钮，将清空当前对话内容并开启一段全新的会话。
  • 搜索历史：在侧边栏搜索框中输入关键字，可以实时过滤并筛选“最近对话”列表。
  • 历史记录管理：新的对话首句会自动截取成为侧边栏的对话标题。鼠标悬停在标题上点击垃圾桶图标即可删除整个历史对话，该操作具有二次确认弹窗防护。
  • 折叠侧边栏：点击侧边栏左上角的图标可收起侧边栏以获得更大的阅读空间；收起后，左上角会出现浮动按钮用于重新展开。

3. 模型与附件管理

  • 切换模型：点击顶部导航栏中央的模型名称（如 ChatGPT 4o），会展开下拉菜单，在此快速切换后台交互的模型。
  • 添加附件：点击输入框左侧的 + 号，可选择上传文件、上传图片，或导入 GitHub 链接。
  • GitHub 导入：在附件菜单点击后，可粘贴项目 URL。系统会进行解析（带状态反馈），成功后会反馈一条系统级解析完成消息。

4. 高级设置面板

点击左下角头像，选择“设置”打开高级配置：

  • 个性化提示词：支持无限添加自定义提示词。
    - 全局提示词在应用内的所有对话中作为系统预设生效。
    - 特定提示词仅覆盖并作用于当前正进行的单次对话。
  • API 智能配置：提供可视化的模型列表管理。您可以通过右上角“添加模型”，不仅支持手动填入 URL 和 Key，还支持一键粘贴 curl 或 Python 代码自动解析。修改后可点击“测试连接”验证。

5. 数据存储与持久化

  • 存储文件夹选择：在“设置 -> 存储设置”中，您可以点击“更改目录”来选择一个专门的文件夹。
  • 持久化保障：将数据存储在应用目录之外，可以确保在更新应用或重新安装时，您的对话信息、上传文件和知识库不会丢失。
  • 自动同步：开启“自动同步”后，每一条对话都会实时备份到您选定的文件夹中。
